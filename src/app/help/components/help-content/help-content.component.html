Catálogo de productos

Registro y consulta del catálogo de productos

El sistema FLEX GYM ADMIN® cuenta con un apartado para el registro y consulta de los productos con los que cuenta el negocio. De esta manera se obtiene un control autimatizado de los productos

Ingresa a registrar un producto

El primer paso para realizar el registro de un producto es ingresar a la sección lateral izquierda en 'Catálogos/Productos/Registrar', cómo se muestra en la primera imagen. De esta manera, estará ingresando a la pagina para registrar un producto.

Registro de producto

Una vez en la página del registro de productos se llenaran los campos de la siguiente manera:

  • Descripción La descripcion del producto, por ejemplo: las caracteristicas principales que contiene ese producto.
  • Marca Se eligirá alguna de las marcas que se encuentren enlistadas y que corresponda con la marca del producto. En el listado se encontrarán todas las marcas que maneja el negocio.
  • Sabor El sabor del producto.
  • Fecha de caducidad El formato de la fecha de caducidad del producto es dd/mm/aaaa siendo 'dd' el día, 'mm' el mes y 'aaaa' el año.
  • Precio de venta Se indica el precio en que se vende el producto a los clientes.
  • Precio de compra Se indica el precio en que se compra el producto a los proveedores.
  • Existencia Ya que todos los productos con los que cuenta el negocio son unidades, se indicará con un numero entero el numero de productos a registrar.
  • Producto máximo Es el tope del número de productos con los que el negocio no debería revasar de ese producto en específico.
  • Producto mínimo Es la cantidad mínima de ese producto con el que el negocio debería contar.

Una vez llenado todos los campos, al dar click en el boton de registrar, el producto será registrado en la base de datos y podra ser visualizado en una consulta general y/o individual.

Consulta general

Para realizar una consulta general de los productos. Se accederá desde el menú de la barra lateral izquierda en 'Catálogos/Productos/Consultar', se puede visualizar esta ruta en la primera imagen de la explicación. Aparecerán todos los registros de los productos con su informacion más importante, en la imagen debajo se muestra uno de los registros el cuál fue creado previamente en la segunda imagen. Del lado derecho del registro, aparecen dos botones: Consultar y Editar.
  • Consultar Se realiza una consulta individual del producto más detallada de la que se muestra en la consulta general.
  • Editar Se muestran los datos del producto para poder ser editados y guardados los cambios en la base de datos.

Consulta individual del producto

Para realizar una consulta individual de un producto se ingresará a la misma ruta que para realizar una consulta general (Véase, Consulta General para mas información). Una vez en la página de consulta general, de click en el botón de Consultar del registro que quiera realizar una consulta individual más detallada. A continuación aparecera una vista como la que se muestra en la siguiente imagen.

Editar un producto

Para realizar la edición de un producto se ingresará a la misma ruta que se hace para realizar una consulta general (Véase, Consulta General para mas información). Una vez en la página de consulta general, de click en el botón de Editar del registro que quiera editar sus datos. A continuación aparecera una vista cómo la que se muestra en la siguiente imagen, se podrán editar los datos del producto y una vez realizados los cambios, de un click en Guardar Cambios para que los cambios que se hicieron tengan efecto.

Movimiento de ventas

Registro y consulta de ventas a clientes

El sistema FLEX GYM ADMIN® cuenta con un apartado para el registro y consulta de las ventas que se realizan a los clientes del negocio. Se lleva a cabo un control automatizado de las ventas del negocio.

Ingresa a registrar una venta

El primer paso para realizar el registro de una venta es ingresar a la sección lateral izquierda en 'Movimientos/Ventas/Registrar', cómo se muestra en la primera imagen de esta sección. De esta manera, estará ingresando a la página para el registro de una venta.

Registro de la venta

Una vez en la página del registro de ventas, se llenarán los campos de la siguiente manera:

  • ID Venta Este campo será llenado automáticamente, es un numero consecutivo único que indica el ID de la venta
  • Fecha Este campo será llenado automáticamente, es la fecha del día actual tomada del sistema operativo.
  • Datos del cliente seleccionado

    Para llenar los datos del cliente, se hará click en el botón con el texto 'Buscar cliente', a continuación, se muestea una ventana como se puede observar en la siguiente imagen, selecciona alguno de los clientes y automáticamente muestra sus datos en el formulario principal del registro de la venta.

  • Datos del empleado seleccionado

    Para llenar los datos del empleado, se hará click en el botón con el texto 'Buscar empleado', a continuación, se muestea una ventana como se puede observar en la siguiente imagen, selecciona alguno de los empleados y automáticamente muestra sus datos en el formulario principal del registro de la venta.

  • Agregar un producto a la venta

    Para agregar un producto a la venta, el primer paso es dar click en el boton 'Buscar producto', a continuación se desplegará una ventana para seleccionar el producto que desee agregar a la venta, automáticamente se llenarán algunos campos del formulario principal del registro de la venta.

    • Precio Se indíca el precio del producto.
    • Cantidad Se indíca el número de productos que se van a agregar a la venta.
    • Agregar producto Para agregar el producto a la tabla sólo de click en el boton 'Agregar producto'.
  • Tipo de venta Donde de tendrá la opción de ser una venta a CRÉDITO o CONTADO.
  • Subtotal Se calcula automáticamente.
  • Descuento Se puede asignar un descuento a desición de la persona que realice la venta.
  • Total Se calcula automáticamente.

El último paso para realizar el registro de la venta es dar click sobre el boton con el texto 'Registrar'
